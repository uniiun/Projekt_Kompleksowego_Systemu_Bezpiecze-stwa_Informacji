# Porownanie konfiguracji MFA

## TOTP
- Plusy: szybkie wdrozenie, brak sprzetu.
- Minusy: podatnosc na phishing i przepisywanie kodu.

## WebAuthn
- Plusy: silna kryptografia, odpornosc na phishing.
- Minusy: wymaga kompatybilnego urzadzenia.

## Wnioski
- Dla kont krytycznych rekomendowane WebAuthn.
- Dla pozostalych kont TOTP jako kompromis koszt/bezpieczenstwo.
